# 2026-02-27

## Remix App - Bullet-Level Cards

Work on transforming the Remix app UI to display individual bullets as cards instead of full note previews:

### Design Change
- User wanted individual bullets as individual cards (like a shuffled deck)
- Each card should show: title, tags, and one bullet content
- Cards should be randomized/shuffled on load
- Clickable tags for filtering

### Implementation
- Rewrote `FeedView.tsx` to create a flat array of bullet cards
- Added shuffle button to randomize order
- Each card displays:
  - Note title as header
  - Tags as badges
  - Single bullet content as body
  - Updated date
- Cards render in a 3-column grid on desktop

### Remix API Fix
- Fixed `/api/remix/route.ts` to handle `bulletIds` instead of `noteIds`
- The API now looks up bullets from the database directly
- Generates content based on selected bullet contents
- Shows which bullets were used in the response

### Result
- 6,259 individual bullet cards now display instead of 221 note cards
- Each card contains one insight that can be selected for remixing
- Shuffled order provides discovery of content

## Files Modified
- `/app/components/FeedView.tsx` - Complete rewrite for bullet cards
- `/app/api/remix/route.ts` - Updated to handle bullet IDs
